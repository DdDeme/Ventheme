// how.to.use
Get your theme into Vencord in 3 steps.
01
Design your theme
Use the editor above to tweak colors, fonts, border radius, effects and more. The live preview updates as you go.
02
Export the CSS
Click Export for Vencord to download your .css file, or copy the CSS directly from the panel.
03
Import into Vencord
Open Discord → Settings → Vencord → Themes → Open Themes Folder. Drop your file in, then enable it. Done!
